🎓 Udemy Clone – Spring Boot Backend Project

This is a full-stack ready backend project built using Java Spring Boot, simulating a Learning Management System (LMS) like Udemy. It includes user registration, role-based access, course management, student enrollment, and secure JWT-based authentication.

🚀 Features

  • User Registration (Admin, Instructor, Student)
  • JWT-based Login Authentication
  • Role-based Access Control
  • Course Creation, Update & Deletion
  • Enroll & Unenroll from Courses
  • View Enrollments and Enrolled Students
  • Secure Spring Security Configuration
  • DTO-Entity Layered Architecture

🌐 REST API Endpoints

🔐 Authentication & Home

  • GET /Welcome/{name} – Welcome message
  • GET /Logged_In – Get current user
  • GET /Login – Login with JWT

👤 User APIs

  • POST /User/Create_Student
  • POST /User/Create_Admin
  • POST /User/Create_Instructor
  • GET /User/Get_By_id/{user_id}
  • GET /User/Get_By_email/{email}
  • DELETE /User/Delete_By_Id/{user_id}
  • DELETE /User/Delete_By_Email/{email}
  • PUT /User/Update_Role/{user_id}/{new_role}

🛡️ Role APIs

  • POST /Role/Create_Admin_Role
  • POST /Role/Create_Student_Role
  • POST /Role/Create_Tutor_Role
  • GET /Role/Get_Role/{role_id}

📚 Course APIs

  • POST /Course/Create_Course/{user_id}
  • GET /Course/Get_Course/{course_id}
  • DELETE /Course/DeleteByID/{course_id}
  • PUT /Course/Update_Duration/{course_id}/{dur}

📝 Enrollment APIs

  • POST /EnrollInCourse/{course_id}/{user_id}
  • GET /Get_Enrollment/{enroll_id}
  • GET /Get_Enrolled_students/{course_id}
  • PUT /UnEnroll_Student/{course_id}/{student_id}

🔐 Security Configuration

  • JWT filter validation
  • Role-based access using Spring Security
  • Stateless auth, CSRF disabled
  • Only selected endpoints are public

▶️ How to Run Locally

git clone https://github.com/your-username/udemy-clone-backend.git
cd udemy-clone-backend
mvn clean install
mvn spring-boot:run
